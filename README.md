# 📊 Enterprise AI Usage (Claude vs Gemini)

## Overview
This project analyzes enterprise adoption of two leading AI platforms — **Claude** and **Gemini** — using a cleaned dataset and interactive Power BI dashboards. It highlights metrics such as user engagement, revenue distribution, satisfaction levels, error rates, latency, and churn risk.

## Problem Statement
Enterprises struggle to identify the most efficient AI platform that balances **cost, performance, and user satisfaction**. This project provides a comparative framework to support strategic decision‑making.

## Dataset
- File: `ai_platform_cleaned_data.csv`
- Key fields:
  - Active_Days_Per_User
  - Age
  - API_Calls_Per_Day
  - Avg_Response_Time
  - Error Rate
  - Revenue
  - Churn Risk

## Tools & Technologies
- Power BI → Dashboard creation & visualization
- Excel/CSV → Dataset preparation & cleaning
- Data Transformation → Ensuring accuracy and usability
